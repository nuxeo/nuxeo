The Nuxeo UI Style Guide

Dear user, welcome. You are here because you care about readability, usability, layout design, CSS classes and user experience. You are right to feel concerned about all these topics, which are the key to a successful platform.

This guide will help you learn how to design the interface to fit your end user's need, discover how to layout your content, explore all the available classes for the different elements, and browse the list of icons embedded in the platform. Following the guidelines described in this guide will make maintenance and upgrade easier when the platform evolves.

We hope that this tool will be the perfect explorer to help you imagine what you can design in your Nuxeo Platform.
