Reusable template to display a form. Accepted parameters: - addForm: boolean stating if a form should be added (optional, defaults to false) - formId: the form id (required if addForm is true) - useAjaxForm: boolean stating if form should be ajaxified (optional, defaults to false) - disableMultiPartForm: boolean stating if non-ajaxified form should *not* use send multipart request - formStyleClass: style class to put on the form (optional) Templating zone to define: - form_template_content: inside the form if a form is defined
