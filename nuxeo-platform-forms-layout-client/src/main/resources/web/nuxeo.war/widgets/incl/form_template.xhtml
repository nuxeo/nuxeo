Reusable template to display a form. Accepted parameters: - addForm: boolean stating if a form should be added (optional, defaults to false) - formId: the form id (required if addForm is true) - useAjaxForm: boolean stating if form should be ajaxified (optional, defaults to false) - eventsQueueId: id for events queue of the ajax form (optional, defaults to the form id suffixed by _eventsQueue) - reRender: ajax elements ids to rerender (optional) - formStyleClass: style class to put on the form (optional) Templating zone to define: - form_template_content: inside the form if a form is defined
