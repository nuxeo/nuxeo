Binds the value given to the layout tag to the sort info list. Sort info presented are generated from the layout, filtering columns that have a non-empty property named 'sortPropertyName' and a property named 'showInSortInfoSelection' resolving to 'true'. Properties to define on the layout: - newSortInfoTemplate (resolving a bare SortInfo instance) - required (boolean) Properties to define on the layout column: - label (for presentation of the column) - a name on the column, used as an identifier
