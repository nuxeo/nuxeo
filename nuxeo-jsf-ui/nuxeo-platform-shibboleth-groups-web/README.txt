Nuxeo Platform Shibboleth Groups Web
====================================

Overview
--------

This addon adds a new widget to search a group in the rights management tab.
You can now explore all defined group by using a tree navigation, and search user
using the autocompletion field.
The new widget appears in the "manage" tab, it overrides the default one.

How to deploy it
----------------

It requires nuxeo-platform-login-shibboleth to be deployed.

Copy the nuxeo-platform-login-shibboleth, nuxeo-platform-shibboleth-groups-web
jars to your Nuxeo 5.4.
